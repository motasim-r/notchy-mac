# Appcast Publishing

`appcast.xml` for Sparkle updates is generated by:

```bash
./native/scripts/notarize_release.sh
```

This script writes:

- `native/appcast/appcast.xml`
- `native/appcast/archives/*.zip` (notarized update archives used by `generate_appcast`)

The app feed URL is configured as:

- `https://motasim-r.github.io/notchy-mac/native/appcast/appcast.xml`

For this to be publicly reachable, enable GitHub Pages on the repository and make sure
`native/appcast/appcast.xml` is pushed to `main`.
